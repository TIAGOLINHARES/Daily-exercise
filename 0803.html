Testando as Listas

Listas de atributos que foram apresentados no estudo de hoje, e armazenados na minha biblioteca css

  • color * cor
  • background-color * cor do fundo
  • font-family: serif * seleciona o tipo de fonte a ser utilizada
  • text-align: righr * alinhamento do texto
  • line-height: 1px * tamanho da altura de cada linha
  • letter-spacing: 3px * tamanho do espaço entre cada letra
  • word-spacing: 5px * tamanho do espaço entre cada palavra
  • text-indent: 30px * tamanho da margem da primeira linha do texto
  • background-image * imagem de fundo
  • bordas e seus atributos
    • border-color: red
    • border-style: solid
    • border-width: 1px
  • comentarios em css * /* comentario aqui */
  • padding: 10px 20px 15px 5px * margem interna
  • margim: * margem externa
    • margin-top
    • margin-right
    • margin-bottom
    • margin-left
  • dimenções
    • height: 300px * altura
    • width: 300px * largura
  • #id * usar como identificação para alguma tag especifica
